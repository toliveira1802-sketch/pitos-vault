# Audi — Originais vs paralelas

Audi compartilha boa parte da cadeia VW AG (Bosch, Mahle, Mann, Hengst, Lemforder, Febi, Pierburg, Continental). Peça "original Audi" é geralmente a mesma do fornecedor em caixa prata.

## OEM suficiente — qualidade idêntica

| Item | OEM preferido |
|------|---------------|
| Filtro óleo | Mahle / Mann / Hengst |
| Filtro ar | Mann / Mahle |
| Filtro combustível | Mahle / Bosch |
| Pastilhas | Textar / Ate / Ferodo |
| Discos | Brembo / Zimmermann / Ate |
| Bateria | Varta AGM / Moura AGM |
| Velas | NGK / Bosch (original spec) |
| Bobina | Bosch / NGK / Eldor |
| Bieleta/pivô | Lemforder / Febi / TRW |
| Bomba d'água | Pierburg / Hepu / INA |
| Correia + tensor | INA / Gates / Continental |
| Lambda | Bosch / NGK |
| Óleo | Castrol Edge 504.00 / Motul 8100 X-clean |
| Fluido DSG | Febi / Pentosin / Original VAG |

## Caixa original **obrigatória**

- **Mecatrônica DSG** — recondicionada ZF/VAG ou nova original.
- **Módulos eletrônicos** (gateway, BCM, ABS, airbag).
- **HPFP** (bomba alta pressão) — paralelo falha rápido.
- **Injetores 2.0 TFSI/TSI** — só Bosch OEM ou VAG original.
- **Sensor Haldex** — codificação via ODIS.
- **Emblemas, grades, molduras visíveis**.
- **Fluido Haldex** — só G 055 175 A2 original.
- **Corrente de distribuição EA888** — IWIS original, nunca paralelo.

## Red flags — não entra na DAP

- Filtros desconhecidos com carvão falso.
- Kit de corrente de origem duvidosa (pular dente = motor fundido).
- Bobinas chinesas — falha rápida em TFSI.
- Pastilhas "esportivas" genéricas em RS/S-Line sem spec.
- Óleo genérico sem certificação VW 504.00/508.00 na embalagem.
- Fluido DSG "compatível" — só G 052 182 (DQ250) ou G 055 529 (DQ500).

## Regra DAP

A gente nunca vende original quando OEM entrega igual. Nunca vende paralelo em item com risco mecânico/eletrônico sério. Cliente vê a caixa, a marca, entende por que a escolha. Peça visível e rastreável sempre.
