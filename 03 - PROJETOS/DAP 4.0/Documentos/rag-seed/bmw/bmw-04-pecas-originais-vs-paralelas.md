# BMW — Originais vs paralelas

A BMW não fabrica quase nada sozinha. Compra de fornecedores OEM (Bosch, Mahle, Hella, ZF, Pierburg, Lemforder, Febi). Peça "original BMW" geralmente é a mesma peça do fornecedor, com caixa azul e preço 2–4x maior.

## Quando faz sentido **OEM (caixa do fornecedor)**

Na maioria dos itens de desgaste, a peça é idêntica. Exemplos:

| Item | OEM preferido |
|------|---------------|
| Filtro óleo | Mahle OX / Mann HU / Hengst |
| Filtro ar | Mahle LX / Mann C |
| Filtro cabine | Mahle LAK / Mann CUK |
| Pastilhas freio | Textar / Ate / Pagid |
| Discos freio | Brembo / Zimmermann / Ate |
| Bateria | Varta AGM (é a BMW) |
| Velas | NGK / Bosch (referência original) |
| Bieleta/pivô | Lemforder / TRW / Febi Bilstein |
| Bomba d'água | Pierburg / Hepu |
| Termostato | Behr / Mahle |
| Bobina | Bosch / Delphi / Eldor |
| Coxim motor | Lemforder / Corteco |
| Sensor lambda | Bosch / NGK |

## Quando **original BMW caixa azul** é o certo

- **Módulos eletrônicos** (DME, ZGW, FRM) — sempre.
- **Sensores proprietários** com software linkado (IBS bateria, sensor volante direção).
- **Peças de segurança estrutural** sob garantia: airbag, cintos, colunas direção.
- **Acabamento visível** (emblemas, grades, molduras) onde paralelo destoa.
- **Correntes de distribuição N20/N26** — risco mecânico alto, OEM Iwis original preferido.

## Red flags — **não aceitamos**

- Filtros "genéricos" sem marca (carvão ativado falso, papel desfiando).
- Bobinas chinesas em motor turbo direto — falha em 3–6 meses.
- Pastilhas genéricas em M-Performance / freio M — fade térmico severo.
- Óleo "equivalente LL-04" sem aprovação BMW listada na embalagem.
- Velas paralelas em N54/N55/S55 — risco de knock.
- Fluido ZF "compatível" — só LifeGuard 8 original ou ZF oficial.

## Regra DAP

Cliente decide com informação. Mostramos peça OEM lado a lado com caixa azul, explicamos a diferença real. Nunca empurramos "original" onde o OEM resolve igual. Nunca poupamos em item crítico de segurança.
