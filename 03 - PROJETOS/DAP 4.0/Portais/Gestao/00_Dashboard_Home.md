# 00 Dashboard Home

> [!info] Objetivo do Módulo
> Tela roteadora inicial (Hub) contendo os principais indicadores de cada departamento (Visão Executiva) servindo como portal para os sub-módulos.

## Requisitos de Interface Visual
- **Cards Coloridos Grid:** 7 Módulos dispostos em grid.
  - **Recursos Humanos:** 12 Mecânicos Ativos (Borda Azul).
  - **Operações:** 24 OS em Andamento (Borda Verde).
  - **Financeiro:** Faturamento Mês R$ 85k (Borda Laranja).
  - **Tecnologia:** 37 Usuários Ativos (Borda Roxa).
  - **Comercial e Marketing:** +23 Novos Clientes (Borda Rosa).
  - **Melhorias:** 8 Sugestões Ativas (Borda Ciano).
  - **Veículos Órfãos:** 156 Aguardando (Borda Laranja).
- Ícones específicos `lucide-react` para cada card.
- Setinha de CTA (Call to Action) para entrar no detalhamento (ArrowRight).
