# 08 RH Analise

> [!info] Objetivo do Módulo
> Relatório Executivo macro da força de trabalho, olhando além da performance e entrando no aspecto orgânico: turnover mental, satisfação laboral e expansão de headcount.

## Requisitos de Interface Visual
- **4 KPIs Superiores:**
  - Azul: 122 TOTAL FUNCIONÁRIOS.
  - Verde: 48 CONTRATAÇÕES (6M).
  - Laranja: 8.5 PERFORMANCE MÉDIA.
  - Roxo: 77% TAXA SATISFAÇÃO.
- **Grade de Gráficos Analíticos (6 Blocos):**
  1. Box 1: **Contratações vs Demissões** - Gráfico de Colunas Agrupadas (Verde/Vermelho) separando por Més (Jan-Jun).
  2. Box 2: **Funcionários por Departamento** - Gráfico de Barras horizontais base (Operações, Vendas, Tecnologia, Marketing).
  3. Box 3: **Nível de Satisfação** - Gráfico de Pizza Clássico dividido em 4 fatias de score.
  4. Box 4: **Performance Média Trimestral** - Evolução (Q1-Q4) com gráfico de linha simples subindo ou nivelando.
  5. Box 5: **Taxa de Turnover Mensal (%)** - Gráfico de linha ondulada suave (Curved Spline Area).
  6. Box 6: **Horas de Treinamento** - Barras ciano agrupando frentes de ensino (Técnico, Liderança, Compliance, Soft Skills).
